# 第 2 阶段：资金闭环

- 注册开户真实落库，自动创建实名/KYC demo 记录、用户安全记录和默认账户。
- 完成 JWT 登录、当前用户、退出登录、账户列表、余额、详情、申请新账户、冻结/解冻、销户接口。
- 完成转账预校验、确认订单、执行转账、 legacy 转账接口、批量预校验/执行、定时转账订单预留。
- 资金变动使用事务、付款账户业务锁、数据库行锁；生成订单和双边流水。
- 实现幂等键重复提交返回原订单结果，不重复扣款。
- 实现账户归属校验，禁止查询或操作他人账户。
- 完成流水分页查询、筛选、详情和 CSV 导出。

验证：后端集成测试覆盖注册、重复注册、登录、越权、转账成功、流水查询、余额不足/风控拒绝、幂等重复提交、并发扣款不为负。
